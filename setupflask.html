Установка Flask

Теперь при активированным виртуальным окружением установим Flask. Для этого в терминале введите pip install Flask. После установки создайте файл app.py и импортируем класс Flask из модуля flask. Также создадим переменную app в которую присвоим класс Flask и в качестве параметра передадим имя нашего модуля с помощью __name__
Так же с помощью конструкции
if __name__ == '__main__":
        app.run(debug=True)
мы говорим что это запускаемый модуль и с помощью метода run у app мы запускаем локальный сервер и для отображения ошибок параметром в метод run добалвяем debug=True(когда вы сделайте деплой на сервер нужно поставить этот параметр в False)
Теперь запустим наш локальный сервер для этого запустим наш файл app.py.Введите в терминале python app.py. После этого в консоли отобразиться локальный сервер по которому мы можем перейти
Если мы перейдем то на странице будет ошибка потому что мы еще не создавали страницы
Добавим нашую первую страницу которая будет выводить на страницу Hello world
Разберем подробнее. С помощью декоратора @app.route(адрес_страницы) мы указываем по какому адресу будет отображаться страница. Так вот '/'' озночает главная cтраница.Далее мы создаем метод index который возвращает нам текст Hello World
Чтобы отключить локальный сервер в терминале нажмите сочитание клавиш CTRL + C
