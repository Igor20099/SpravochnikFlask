Создание виртуального окружения

Для начала создадим папку flask_project(вы можете назвать как угодно) и в этой папке создадим виртуальное окружение. Для этого в терминале введите команду python -m venv название_окружения в моем случае это будет python -m venv env
далее активируем виртуальное окружения для этого нужно в терминале ввести .\название_окружения\Scripts\activate в моем случае это будет .\env\Scripts\activate. После активации у вас в консоли в начале строки в скобках должно появиться название вашего окружения
Чтобы деактивировать виртуальное окружение введите в терминале deactivate. Нам нужно чтобы виртуальное окружение было активно для того чтобы отделить устанавливаемые модули от глобальных модулей Python
